# 公共模块

### 目录结构

```
llz-cloud-common
│
└─── llz-core-annotation - 封装注解
│
└─── llz-core-auth - 封装jwt
│
└─── llz-core-common - 封装通用工具
│
└─── llz-core-job - 封装xxl-job
│
└─── llz-core-launcher - 封装启动器(扩展springApplication启动过程)
│
└─── llz-core-mybatis - 封装mybatis-plus
│
└─── llz-core-sentinel - 封装sentinel熔断
│
└─── llz-core-test - 适配自定义扩展的启动器而来的单元测试注解
│
└─── llz-core-web - web能力

```
